## 基础组件（被依赖，不可删除）

| 组件 | 用途 |
|------|------|
| Avatar | 用户头像，支持 initials 回退、多种尺寸/形状 |
| Badge | 数字徽标，支持上限、dot 模式、多色 |
| Breadcrumb | 面包屑导航，支持点击跳转 |
| DataTable | 通用数据表格，支持排序、分页、搜索、选择 |
| Drawer | 侧滑抽屉，左右分栏布局（60/40），含 header/footer |
| DropdownMenu | 下拉菜单，支持图标、危险操作标记 |
| EmptyState | 空状态占位，支持操作按钮 |
| Pagination | 分页控件，支持快速跳转和页码切换 |
| StatusBadge | 状态徽章，25+ 种预设状态，自动映射标签+颜色 |
| UserSelect | 用户选择弹窗，支持搜索和过滤 |

## 样式参考组件（参考结构模式，不复用 props）

### 审批类

| 组件 | 展示的模式 | 参考价值 |
|------|-----------|----------|
| ApprovalConfig | 多步骤审批流配置面板 | 流程配置 UI 的结构模式 |
| ApprovalFlow | 审批进度抽屉（步骤条 + 审批操作） | 步骤进度 + 操作按钮的抽屉布局 |
| ApprovalTimeline | 审批节点时间线（单节点展示） | 时间线节点的视觉模式 |

### 卡片/列表类

| 组件 | 展示的模式 | 参考价值 |
|------|-----------|----------|
| CardHeader | 卡片头部（标题 + 副标题 + 状态 + 菜单） | 卡片头部标准模式 |
| DualItemCard | 阴阳项目卡片（客户项 + 生产项） | 一对多/多对一关系展示模式 |
| OrderItemCard | 订单项卡片（状态 + 操作 + 阴阳标记） | 列表项卡片模式 |
| PaymentCard | 付款记录卡片（金额 + 状态） | 金额展示卡片模式 |

### 工作流/面板类

| 组件 | 展示的模式 | 参考价值 |
|------|-----------|----------|
| InvoiceProcess | 发票处理面板（选择列表 + 操作） | 选择+处理的工作流面板 |
| PermissionConfig | 权限矩阵（Checkbox 网格） | 权限配置面板模式 |
| SearchFilterBar | 搜索栏 + 筛选下拉 + 新建按钮 | 列表页顶部工具栏模式 |
| SelectionMode | 批量选择模式（全选 + 批量操作） | 批量操作交互模式 |

### 通知/日志类

| 组件 | 展示的模式 | 参考价值 |
|------|-----------|----------|
| NotificationCard | 单条通知卡片（图标 + 已读状态） | 通知卡片视觉模式 |
| NotificationList | 通知列表（分组 + 查看全部） | 通知列表布局 |
| OperationLog | 操作日志（时间线式） | 操作记录展示模式 |
| LoginLog | 登录日志表格（筛选 + 搜索 + 表格） | 日志类页面的标准模式 |
