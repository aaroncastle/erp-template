## 布局优势总结

| 特点 | 说明 |
|------|------|
| **单页操作** | 所有操作在列表页完成，无需跳转 |
| **功能散落** | 按钮放在卡片上，直观可见 |
| **无侧边栏** | 顶部导航 + Tab，节省空间 |
| **C 端体验** | 类似淘宝订单、滴滴行程 |
| **权限隐藏** | 无权限的按钮直接不显示 |
| **响应式** | 移动端自适应 |
| **抽屉详情** | 查看详情不离开列表 |
| **状态筛选** | Tab 快速切换不同状态 |
